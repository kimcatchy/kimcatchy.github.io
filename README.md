# 🕳️ Burrow

Welcome to **Burrow**, my personal digital archive. I started this blog as a dedicated space to organize and solidify everything I learn while studying programming. It serves as a continuous record of my growth, capturing the evolution of my thoughts, the development processes of my projects, and the detailed steps I take to solve complex technical problems.

# 🚀 Organizing the Journey
This blog is born from the need to transform fleeting lessons into lasting knowledge. As I encounter new concepts and technologies during my studies, I use this space to structure and document them. Writing helps me refine my understanding, turning abstract ideas into concrete references I can always return to.

# 🛠️ Recording the Process
Development is more than just writing code; it's about the decisions, challenges, and solutions encountered along the way. I use Burrow to document the **why** and **how** behind my projects. By archiving the specific problems I face and the steps taken to resolve them, I build a searchable database that aids my future self and potentially helps others facing similar hurdles.

# 🌿 The Philosophy
In an era of bloated web experiences, Burrow stands for simplicity and speed. This blog is designed to be a frictionless environment:
- **Focus on Clarity**: A dense, distraction-free UI that keeps the spotlight on technical details and learning points.
- **Natural Workflow**: Managed through Notion, allowing for a seamless transition from thought to published post via automation.

<summary>🛠️ Repository & Technical Information</summary>

This repository contains the core **Astro** blog framework.

## Features
- **Framework**: [Astro 6.x](https://astro.build/) with the new Content Layer.
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) & [shadcn/ui](https://ui.shadcn.com/).
- **Search**: Fuzzy search powered by [Fuse.js](https://www.fusejs.io/).
- **Sync**: Integrated with a custom Notion-to-Markdown service.

## Getting Started
```bash
# Clone and install
git clone https://github.com/your-username/blog.git
cd blog/astro
npm install

# Run locally
npm run dev
```
